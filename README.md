# Thread Hub iOS
## 概要
匿名掲示板アプリ「Thread Hub」のiOS版。
ユーザーがスレッドを作成し、他のユーザーがそれにコメントを追加していく仕様です。

## フレームワーク・ライブラリ
- SwiftUI
- [Firebase](https://github.com/firebase/firebase-ios-sdk)
